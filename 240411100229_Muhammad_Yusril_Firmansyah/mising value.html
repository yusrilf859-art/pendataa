1. Missing Values Imputation dengan WKNN

Missing value adalah data yang hilang dalam dataset. Salah satu metode untuk mengatasinya adalah Weighted K-Nearest Neighbor (WKNN). Metode ini mencari data terdekat lalu menghitung nilai pengganti menggunakan bobot jarak.

Contoh Dataset

Tinggi Berat
170 65
165 60
180 ?
175 70

Langkah Perhitungan WKNN

1. Hitung jarak

d = |180 - tinggi|

  • 170 → 10
  • 165 → 15
  • 175 → 5

2. Hitung bobot

w = 1 / d

  • 0.1
  • 0.066
  • 0.2

3. Hitung nilai imputasi

Berat ≈ 66.8 kg

jadi data yang hilang = 66.8kg

2. Normalisasi Data

Normalisasi data adalah proses mengubah nilai data ke skala tertentu agar perbedaan antar fitur tidak terlalu besar.

Jenis Normalisasi

    1.Min-Max Normalization

    rumus

    (x - min) / (max - min)

    contoh data

    150, 160, 170, 180

    min=150
    max=180

    normalisai untuk 170

    (170-150)/(180-150)
    = 20/30
    = 0.67

    Z-Score Normalization

    rumus

    z = (x - mean) / std

    contoh:
    mean=165
    std=10

    (170 - 165) / 10 = 0.5

    Decimal Scaling

    rumus

    x / 10^j

    contoh

    987 → 0.987
